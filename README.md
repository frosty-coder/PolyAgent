# PolyAgent

**Cross-Platform Modular Security & Research Framework**

PolyAgent is a cross-platform, modular framework for security researchers, penetration testers, and developers. Manage and control agents across **Windows, Linux, and Android** with minimal setup. PolyAgent focuses on **flexibility, modularity, and secure communication**, giving you full control over agent generation.

## Features
- **Multi-Platform Support:** Windows (Python, C++, PS1, BAT), Linux (Python, Go), Android (Java/Kotlin APK)
- **Modular Design:** Easily add custom plugins for each OS
- **Secure Communication:** AES + RSA encryption for all client-server communication
- **Task Queue:** Assign commands to one or multiple clients
- **Real-Time Control:** Optional WebSocket support for live tasks
- **User-Friendly Builder:** Generates client folders with config and compilation instructions
